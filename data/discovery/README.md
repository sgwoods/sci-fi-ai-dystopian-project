# Discovery Data

This directory holds the widening-search discovery pool for potential quote
source works.

## Main Files

- `ai-dystopia-title-discovery.json`
  Title-level discovery queue for books, stories, films, games, and shows that
  may yield quotable dystopian-AI material.
- `ai-dystopia-source-registry.json`
  Growing list of places to look, with type, cadence, and trust notes.
- `ai-dystopia-query-library.json`
  Reusable search patterns and ideation prompts for future quote hunts.
- `ai-dystopia-followup-watchlist.json`
  Recurring author, title, and community lanes worth revisiting.
- `ai-dystopia-author-priority.json`
  Wider ranked author lane for deliberate literary expansion.
- `ai-dystopia-author-top10.json`
  Focused first-pass author mining set for the current widening cycle.

## Purpose

The discovery pool is intentionally broader than the quote review board.
Its job is to help us widen the search, remember where to look, keep search
language that works, and track promising works before individual quotes are
mined and normalized.
