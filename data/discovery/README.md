# Discovery Data

This directory holds the widening-search discovery pool for potential quote
source works.

## Main Files

- `ai-dystopia-title-discovery.json`
  Title-level discovery queue for books, stories, films, games, and shows that
  may yield quotable dystopian-AI material.

## Purpose

The discovery pool is intentionally broader than the quote review board.
Its job is to help us widen the search and track progress toward the current
candidate-goal count before individual quotes are mined and normalized.
